&nbsp;BlinkIT Grocery Sales - EDA Project



&nbsp;Project Overview



This project performs Exploratory Data Analysis on the BlinkIT grocery dataset to identify sales trends and business insights.

Tools Used

\- Python

\- Pandas

\- NumPy

\- Matplotlib

\- Seaborn



Key Analysis



\- Data cleaning and missing value handling

\- Outlier detection

\- Univariate, Bivariate, and Multivariate analysis

\- Correlation heatmaps and visualizations



Insights



\- Medium outlets in Tier 3 locations show highest sales

\- Fruits \& Vegetables and Snacks are top categories

\- Item weight has no impact on sales
